- 使用 Python 内置的 random 模块

混合熵源随机算法 (entropy_random.py)：
- 结合多个随机源（系统时间、硬件事件、密码学随机等）
- 优点：随机性更好，可以在现场收集额外的随机性
- 缺点：实现较复杂，需要更多系统资源
- 适用场景：大型抽奖活动，需要现场演示随机性
